 no ip redirects
 ip mtu 1400
 ip nhrp authentication OTUS
 ip nhrp map multicast dynamic
 ip nhrp network-id 1001
 ip tcp adjust-mss 1360
 tunnel source Loopback1
 tunnel mode gre multipoint
